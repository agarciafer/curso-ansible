Playbook para instalar en caso de que el servidor sea centos instale los paquestes:
mariadb-server
php
httpd
mc

Se requiere realizarlo con una variables llamada paquetes

---
- name: Install docker Centos
  gather_facts: No
  hosts: clientes
  user: root
  vars:
    paquetes:
	 - php
	 - mc
	 
  tasks:
    - name: paquetes
      yum:
        name: "{{  item  }}"
        state: latest
      loop:  "{{ paquetes }}"
   when: ansible_facts['distribution'] == 'CentOS'
